## grid-lines
### grid-column-start/end
```css
.one {
    grid-column-start: 1;
    grid-column-end: 4;
}
// same as 
    grid-column: 1/4;
```
this `one` line starts with line 1, go on to the line 4.  
this grid lines are relative.
